# ak84multimeter

High-performance and lightweight JavaScript library. Ohms law calculator: voltage, current, resistance, power and voltage divider.

## Getting started

```javascript
// volts(amperes, ohms)
volts(2, 5) == 10

// amperes(volts, ohms)
amperes(5, 2) == 2.5

// ohms(volts, amperes)
ohms(12, 2) == 6

// watts(volts, amperes)
watts(12, 10) == 120

// voltageDivider(volts, r1, r2)
voltageDivider(10, 500, 500) == 5
```
